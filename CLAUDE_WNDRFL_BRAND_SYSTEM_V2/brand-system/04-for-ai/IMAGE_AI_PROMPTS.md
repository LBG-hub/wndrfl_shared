## Photography Style Prompts

### Base Style Anchor
Append this to any wndrfl photography prompt to enforce visual direction:

```
cinematic landscape photography, golden hour light, high contrast, moody atmospheric, warm to neutral color temperature, large scale environment, human figure small in vast natural setting, analog film aesthetic, no saturation grading, no HDR processing
```

### Landscape / Aerial

```
aerial photography of rugged coastline at golden hour, distant ocean horizon, warm tones, cinematic, high contrast, no people, vast empty landscape --ar 4:5 --style raw
```

```
low angle wide shot of desert mesa at dusk, single human figure at distance, warm neutral tones, cinematic composition, large format photography aesthetic --ar 4:5
```

```
aerial view of pacific coast cliffs and ocean, morning fog burning off, warm light, high contrast, no text, no graphics, photorealistic --ar 16:9
```

## Graphic Element Generation Prompts

Use these to generate new graphic elements in the wndrfl style. Describe only in formal terms.

### Dot Trail Style

```
hand-drawn ink illustration, irregular cluster of small circular dots of varying sizes, organic asymmetric arrangement, no fill, black ink on white background, etching-style, varied dot density, isolated composition, no background, no color
```

```
hand-drawn ink dots, narrow vertical column of circular marks, varied diameter, irregular spacing, organic placement, black on white, isolated on transparent background
```

### Line Cluster Style

```
hand-drawn ink illustration, cluster of flowing parallel organic strokes, curved and varied weight, etching technique, black ink, isolated composition, no color, no background, asymmetric arrangement
```

```
hand-drawn ink lines, single thin horizontal band of parallel organic strokes, varied density, natural irregularity, black on white, wide aspect ratio composition
```

### Pattern Style

```
dense hand-drawn repeating pattern, layered undulating horizontal organic forms, etching-style rendering, black ink, wide landscape format, full coverage composition, no color, uniform tone
```

## Prohibited Prompt Language

Never include in any wndrfl-related image prompt:

- tribal, ethnic, indigenous, folk, native, cultural, heritage, traditional, artisanal (when implying cultural origin)
- Any reference to a specific culture, tradition, or geographic artistic tradition
- luxury, premium, exclusive, bespoke
- vibrant, colorful, saturated, neon, bright
- corporate, professional, clean modern (these imply off-brand aesthetic)
- Any color name outside the brand palette

## Notes on AI-Generated Content

AI-generated photography is an acceptable production substitute when licensed stock or original photography is unavailable. All AI-generated content must comply with the same color, composition, and subject matter rules as original photography.

AI-generated graphic elements are supplementary only — the 15 dot, 9 line, and 2 pattern SVGs from the commissioned set are the authoritative source. Use AI-generated elements only when the commissioned set does not cover the specific composition need, and only after confirming with the brand team.
