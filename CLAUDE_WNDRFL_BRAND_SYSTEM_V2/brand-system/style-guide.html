07 — Rules
Always
  • Four colors only — #D3CCC2 · #DDD7CE · #E5E1DF · #000000
  • Two typefaces only — Garamond headings, Pressura body
  • wndrfl always lowercase in logo contexts
  • SVG assets inline in HTML — GitHub raw URLs are CORS-blocked
  • Crop viewBox to artwork bounds — never 0 0 4500 4500
  • Graphic elements: black on light, white on photography
  • Lockups contain logotype — never place logo above a lockup
  • Describe graphic elements in formal terms only — dot, line, stroke, cluster
Never
  • Gradients or off-palette colors — no gold, green, magenta, neon
  • Third typeface in any context
  • Symbol in Instagram posts or stories
  • Drop shadows, glows, bevels, or any effects on assets
  • Rotate the logo or symbol
  • All-caps headlines in primary voice
  • Exclamation points in primary copy
  • Cultural, geographic, or heritage language for graphic elements
